# greenfield-hospital-file-server
Design and implementation of a secure department file-access system for greenfield hospital using ubuntu server

## Overview
This project demonstrates the implementation of a secure departmental file-access system using Ubuntu Server 24.04 LTS.

## Skills Demonstrated
- Linux Administration
- User and Group Management
- SSH Configuration
- UFW Firewall
- File Permissions
- SGID
- Access Control

## Tasks Completed
- Configured the hostname
- Configured SSH
- Configured UFW
- Created users and groups
- Secured departmental directories
- Tested access control
